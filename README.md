# WebDash
Privacy-first macOS desktop in your browser—local files, smooth animations, pro UI.
## Features
- Full window manager (drag, resize, minimize)
- Finder + local IndexedDB VFS (create/delete/drag-drop)
- Terminal, Browser, Image Viewer, Text Editor
- Liquid glass design, Dock magnify, MenuBar
- Settings (wallpaper, dock, motion)
## Usage
- Refresh preview to sync
- Deploy: Top-right button
- Export GitHub: Customize forever
Built on Cloudflare Workers + React.
